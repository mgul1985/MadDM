https://feynrules.irmp.ucl.ac.be/attachment/wiki/LQDM/LQDM_UFO.tar.gz
>> unzip LQDM_UFO.tar.gz

>> ./bin/mg_aMC
>> import model LQDM_UFO
>> generate g g > ~chi0 mu+ c ~chi0 mu- c~
>> output template_LQDM_v0
>> launch template_LQDM_v0
run as normal
